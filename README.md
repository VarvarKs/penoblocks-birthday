# Подарок на день рождения: пеноблоки для стен 🧱

Небольшой одностраничный сайт-приглашение: вместо классических подарков можно «подарить пеноблоки» для стен в новой квартире без стен.

## Что это за страница

- Минималистичный лендинг с тёплым, слегка хиппи-вайбом.
- Визуализация стены из пеноблоков — каждый блок символизирует вклад в будущий уют.
- Управление количеством блоков:
  - кнопки «+» и «–»;
  - слайдер.
- Автоматический пересчёт суммы (1 блок = 400 ₽).
- Кнопка «Подарить пеноблоки» показывает благодарность и кнопку с заглушкой-ссылкой на перевод.

## Как запустить локально

1. Склонировать репозиторий:

   ```bash
   git clone git@github.com:VarvarKs/penoblocks-birthday.git
   cd penoblocks-birthday
   ```

2. Скопировать `config.example.js` в `config.js` и указать в нём ссылку на сбор (перевод):

   ```bash
   cp config.example.js config.js
   ```

   В `config.js` задаётся переменная `window.PAYMENT_URL` — сюда подставляется ваша ссылка на сбор средств (карта, СБП, Paymo и т.п.). Файл `config.js` в репозиторий не коммитится (см. `.gitignore`).

3. Открыть файл `index.html` в браузере:

   - двойным кликом по файлу в файловом менеджере, **или**
   - через любую локальную статику/HTTP‑сервер.

Библиотеки и фреймворки не используются — только чистые `HTML`, `CSS` и `JavaScript`.
